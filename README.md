# Drone Viewer

Viewing application for drone project.

### Building

The viewer is built with cmake:

```
./build.sh
```

### Running

The viewer can be run either with Linux or Cygwin, both with X11:

##### Linux (Arch/Manjaro)

`./build/main`

##### Cygwin

`startxwin ./build/main`
